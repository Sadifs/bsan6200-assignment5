PROJECTS

Paris Agreement Tracker (BSAN 6070 Final Project)
Built a machine learning classifier predicting whether countries align with Paris Agreement climate targets using GHG emissions data from 1950 to 2024 across 39 major emitters. Compared Logistic Regression (0.900 ROC-AUC), Gradient Boosting (0.885 ROC-AUC), and Random Forest; selected Random Forest after GridSearchCV: 0.933 ROC-AUC, 84.6% accuracy. Key finding: nearly all on-track countries are lower-income nations with low per-capita emissions. Built a live Streamlit dashboard with global mapping, SHAP explainability, and interactive prediction tools. This project directly applies MEnvSc domain knowledge to a real climate policy question.
GitHub: github.com/Sadifs/BSAN6070-Final

Ask My Resume — RAG Chatbot (BSAN 6200 Final Project)
Built a Retrieval-Augmented Generation chatbot answering recruiter questions about professional background. Pipeline: sentence-transformers embeddings, cosine similarity retrieval, Qwen/Qwen2.5-7B-Instruct via HuggingFace API. Achieved 4.2/5 average quality score across 10 evaluation questions with zero hallucinations. Deployed on Streamlit Cloud.
GitHub: github.com/Sadifs/bsan6200-assignment5

Naive Bayes Sentiment Analysis Pipeline
Text classification pipeline for sentiment analysis using Naive Bayes. Applied NLP preprocessing: tokenization, stopword removal, TF-IDF vectorization. Evaluated on benchmark datasets for accuracy and F1 score.
